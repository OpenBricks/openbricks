NYTTIGA KOMMANDON
m : Dolj/Visa Menyn
o : Visa OSD Timern
s : Byt TV-ut
v : Dolj/Visa Undertextning
Left : Avbryt / Ga Bakat
Right : Ok / Ga Framat
Down : Ga Bakat 1 minute
Up :Ga Framat 1 minute
+/- : Ljud Korrigering +/- 100ms
O/9 : Volym +/- 1

COMMANDS IN FILE SELECTION
d  : set destination directory
c  : copy to destination
r  : ask to remove file
o  : confirm file removal
p  : play the entire directory
l  : recursively view the entire directory
u  : append to playlist
