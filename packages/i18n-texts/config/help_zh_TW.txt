USEFUL COMMANDS
m  : 顯示/隱藏 選單
o  : 顯示 OSD Timer
s  : 切換至 TV-Out
v  : 顯示/隱藏 字幕
Left : 取消 / 向後快轉
Right : 確定 / 向前快轉
Down : 向後快轉 1 minute
Up : 向前快轉 1 minute
+/-  : 聲音 延遲 +/- 100ms
O/9 : 音量 +/- 1

COMMANDS IN FILE SELECTION
d  : 設定目的地目錄
c  : 拷貝至目的地
r  : 詢問移除檔案
o  : 確認檔案移除
p  : 播放/觀看整個目錄
l  : 重復觀看整個目錄
u  : append to playlist
