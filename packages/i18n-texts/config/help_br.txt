COMANDOS UTEIS
m  : Mostrar/Esconder o menu
o  : Mostrar o tempo de reproducao
s  : Ativar saida para TV
v  : Mostrar/Esconder legenda
Esquerda : Cancela / Retorna
Direita : Ok / Avancar
Para Cima : Rebobinar 1 minute
Para Baixo : Avancar 1 minute
+/-  : Atraso do audio +/- 100ms
O/9 : Volume +/- 1

COMANDOS DE SELECAO DE ARQUIVOS
d  : configura diretorio de destino
c  : copiar para o destino
r  : pedir para remover arquivo
o  : confirmar remocao de arquivo
p  : reproduzir todo diretorio
l  : reproduzir todo o diretorio e os subdiretorios
u  : append to playlist
