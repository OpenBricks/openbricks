TECLAS ÚTILES
m  : Mostrar/Ocultar el Menú
o  : Mostrar temporizador OSD
s  : Conmutar salida a TV/CRT
v  : Mostrar/Ocultar Subtítulos
Izquierda : Cancelar / Atrás
Derecha : Aceptar / Adelante
Abajo : Atras 1 minuto
Arriba : Adelante 1 minuto
+/-  : Retardo del sonido +/- 100ms
*// : Volumen +/- 1

TECLAS EN SELECCIÓN DE ARCHIVOS
d  : establecer directorio de destino
c  : copiar a destino
r  : borrar archivo
o  : confirmar borrado de archivos
p  : reproducir directorio completo
l  : ver recursivamente el directorio completo
u  : añadir a la lista de reproducción
i  : grabar flujo de datos
